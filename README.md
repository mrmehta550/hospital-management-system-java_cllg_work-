# 🏥 Hospital Management System (Java)

A simple and efficient **Hospital Management System** built using **Core Java (OOP concepts)**.
This project helps manage patients, doctors, and appointments in a structured way.

---

## 🚀 Features

* 👨‍⚕️ Add & View Doctors
* 🧑‍🤝‍🧑 Add & Manage Patients
* 📅 Book Appointments
* 📋 View Appointment Records
* 🧠 Clean OOP-based design

---

## 🛠️ Tech Stack

* **Language:** Java
* **Concepts Used:** OOP (Encapsulation, Inheritance, Abstraction)
* **Tools:** VS Code / IntelliJ / Eclipse
* *(Optional Upgrade: MySQL + JDBC)*

## ⚙️ How to Run

### 1️⃣ Clone the Repository

```
git clone https://github.com/mrmehta550/hospital-management-system-java_cllg_work-.git
```

### 2️⃣ Navigate to Folder

```
cd hospital-management-system-java_cllg_work-
```

### 3️⃣ Compile the Code

```
javac *.java
```

### 4️⃣ Run the Program

```
java Main
```
